# 🚀 Meu Roadmap de Carreira: Analista de Dados

Este repositório contém o projeto prático desenvolvido durante o bootcamp da [DIO (Digital Innovation One)](https://www.dio.me/). O objetivo do projeto foi utilizar Engenharia de Prompts e Inteligência Artificial para gerar um planejamento de carreira personalizado na área de tecnologia.

Este projeto foi inspirado e estruturado a partir do repositório base da DIO:
👉 [digitalinnovationone/copilot-prompts](https://github.com/digitalinnovationone/copilot-prompts)

## 🤖 Inteligência Artificial Utilizada
* **Modelo:** Gemini (Google)
* **Objetivo:** Atuar como um Especialista em Carreira Tech, recebendo dados de perfil e disponibilidade para gerar um roadmap acionável e focado nas demandas de mercado.

---

## 🎯 O Objetivo
Criar um plano de estudos e desenvolvimento de 90 dias para a transição/início de carreira como **Analista de Dados (Data Analyst)**, considerando uma disponibilidade de 14 a 21 horas semanais.

## 🧠 Prompts Utilizados
Para chegar ao resultado ideal, a interação com a IA foi dividida em etapas. Você pode conferir os prompts exatos que utilizei na pasta `/prompts`:
* [Prompt 1: Coleta de Dados (Agent 1)](#)
* [Prompt 2: Geração do Roadmap e Refinamento de Portfólio (Agent 2)](#)

> **Nota de aprendizado:** O refinamento do prompt final incluiu a exigência de versionamento (Git/GitHub) nas primeiras semanas e um projeto prático focado em regras de negócio (Retail Intelligence) para gerar maior impacto visual em processos seletivos.

## 🗺️ O Resultado: Roadmap de 90 Dias
O plano gerado cobre desde os fundamentos até a preparação para entrevistas. Os principais pilares são:

1.  **Mês 1:** Lógica com Python e domínio absoluto de Git e GitHub.
2.  **Mês 2:** Estruturação de Bancos de Dados, SQL avançado (JOINs) e conexão com Python.
3.  **Mês 3:** Construção do portfólio de impacto e preparação de currículo/entrevistas.

## 💻 Projeto de Portfólio Definido
**Dashboard de Otimização de Vendas e Previsão de Estoque (Retail Intelligence)**
Simulação de um cenário real de varejo para analisar histórico de vendas, lucro por categoria e churn de clientes, utilizando **Python, SQL e ferramentas de visualização**.

## 🛠️ Tecnologias e Ferramentas Mapeadas
* Python (Pandas, NumPy)
* SQL (PostgreSQL)
* Git & GitHub
* Power BI / Looker Studio
